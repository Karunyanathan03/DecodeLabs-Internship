# Data Analytics Internship Project

## Overview

This repository contains the complete Data Analytics Internship Projects completed as part of the Decodelabs Internship Program.

The project includes:

* Data Cleaning and Preparation
* Exploratory Data Analysis (EDA)
* SQL Data Analysis using Oracle SQL*Plus

---

# Tools Used

* Microsoft Excel
* Oracle SQL*Plus
* GitHub

---

# Dataset Used

E-commerce Order Dataset

---

# Project 1 – Data Cleaning & Preparation

## Objective

The objective of this project is to clean the raw dataset and prepare it for analysis.

## Tasks Performed

* Removed duplicate records
* Handled missing values
* Corrected data formats
* Validated data entries
* Standardized dataset structure

---

# Project 2 – Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to analyze the cleaned dataset using Excel pivot tables, charts, and statistical analysis.

## Analysis Performed

* Product Revenue Analysis
* Payment Method Analysis
* Order Status Analysis
* Monthly Sales Analysis
* Referral Source Analysis

## Charts Created

* Product Revenue Bar Chart
* Monthly Sales Line Chart
* Payment Method Pie Chart
* Order Status Column Chart
* Referral Source Bar Chart
